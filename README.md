Baker Bettie's Master Recipe Tracker

A self-hosted, interactive learning dashboard for following Baker Bettie's Baking Fundamentals.

What is this?

This is a lightweight HTML application that organizes Baker Bettie's free online content into a structured "Master Recipe" curriculum, similar to her paid book. It allows you to:

Track your progress through different modules (Muffins, Biscuits, Yeast, etc.).

Direct access to the "Science" article, the "Technique" video, and the "Master Recipe" for each topic.

Save your progress locally.

How to use

This project is designed to be hosted on GitHub Pages.

Visit the site: [Link to your GitHub Pages URL will go here]

Start Learning: Click the links to watch videos or read articles.

Check off items: Click the cards to mark them as complete.

Syncing: Use the "Backup" and "Restore" buttons in the sidebar to move your progress between your laptop and iPad.
